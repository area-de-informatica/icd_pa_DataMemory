Bienvenido al OVA de Evaluación de Memoria

Este Objeto Virtual de Aprendizaje (OVA) está diseñado para ayudar a entender y evaluar diferentes aspectos de la memoria humana mediante técnicas de Inteligencia Artificial.

¿Qué es la memoria?

La memoria es una función cognitiva fundamental que nos permite codificar, almacenar y recuperar información. Es esencial para el aprendizaje, la toma de decisiones y prácticamente todas las actividades cotidianas.

¿Qué puedes hacer con este OVA?

  • Cargar datos de pruebas de memoria en formato CSV
  • Generar datos de ejemplo para análisis
  • Visualizar resultados mediante gráficos interactivos
  • Explorar un modelo de IA para clasificación de capacidad de memoria
  • Hacer predicciones con datos personalizados

Cargar Dataset

Carga un archivo CSV con datos de pruebas de memoria o genera datos de ejemplo para analizar.

Cargar Archivo CSV

El archivo CSV debe contener las siguientes columnas:

  • participante_id: Identificador del participante
  • edad: Edad del participante
  • nivel_educacion: Nivel educativo (1-3)
  • memoria_inmediata: Puntuación (0-10)
  • memoria_trabajo: Puntuación (0-10)
  • memoria_visual: Puntuación (0-10)
  • tiempo_reaccion: Tiempo en milisegundos
  • precision_respuestas: Porcentaje (0-100)
  • atencion_sostenida: Puntuación (0-10)
  • fatiga_cognitiva: Escala Likert (1-5)
  • categoria_memoria: Etiqueta ('bajo', 'medio', 'alto')

Explicación del Modelo

Este OVA utiliza un modelo de Random Forest para clasificar la capacidad de memoria en tres categorías: bajo, medio y alto.

¿Qué es Random Forest?

Random Forest es un algoritmo de aprendizaje automático que crea múltiples árboles de decisión y los combina para obtener una predicción más precisa y estable. Es especialmente útil para problemas de clasificación con múltiples variables.

¿Qué debe incluir un dataset de tests de memoria?

Para realizar un análisis completo de la capacidad de memoria, un dataset debería incluir:

  • Datos demográficos: Edad, nivel educativo, género, etc.
  • Medidas de memoria inmediata: Capacidad para recordar información recién presentada
  • Medidas de memoria de trabajo: Capacidad para manipular información mientras se mantiene en mente
  • Memoria visual: Habilidad para recordar información presentada visualmente
  • Tiempos de reacción: Velocidad de procesamiento y respuesta
  • Precisión: Porcentaje de respuestas correctas
  • Factores contextuales: Nivel de fatiga, atención sostenida, etc.
